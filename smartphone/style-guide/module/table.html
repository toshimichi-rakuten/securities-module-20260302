※ 当モジュールは下層ページのみ有効となります

  • セル幅用汎用クラスは下記で用意しています。（5%刻み）
    .s1-tbl--width-10～.s1-tbl--width-60
  • 縦要素の指定方法は下記クラスで可能です。
    • va-b : vertical-align:bottom
    • va-m : vertical-align:middle
  • ボーダー用クラスを追加しました。
    • tbl-bdr-top : 上
    • tbl-bdr-right : 右
    • tbl-bdr-bottom : 下
    • tbl-bdr-left : 左
    ※ 非表示用クラス
    • tbl-bdr-none--none : 上下左右
  • 手数料ページで多用しているので参考にどうぞ
